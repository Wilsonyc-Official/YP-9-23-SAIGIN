# Конспект Функции
## Функции и параметры
При обработке при разработке большинства приложений

Их разделяют на функциональные модули

В C# приложение состоит из классов, которые содержит именованные блоки кода

Называемые метомадми

Метод(Функици) - это часть класса, которые может осуществлять действия или вычислять значения

## Использование методов

Метод - это набор операторов, которые выполняются вместе для создания метода

необходимо задать его имя, определить список параметров и тело метода

Для вызова метода другого класса, необходимо чтоб он был объявлен как public

Вызов осуществляется по имени класса и метода

Если метод не void, то необходимо вернуть значения соответствующего типа

Оператор return остонавливает выполнение метода и передает отправление вызвавшему данному методу оператору

Каждый метод имеет набор своих локальных переменных, они видны только в нём и при завершении работы метода уничтожаются

Для того чтобы переменные были видны в нескольких методах класса необходимо объявить их полями вне метода но внутри класса
